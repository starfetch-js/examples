# SIMBAD Field Object Types

[Open this example in StackBlitz](https://stackblitz.com/github/starfetch-js/examples?file=04-simbad-field-object-types%2FREADME.md)

Retrieve known SIMBAD objects around a small sky region with object type,
identifier, and coordinates.

Service: SIMBAD TAP preset, `simbad`.
Table: `basic`.

Run:

```sh
node 04-simbad-field-object-types/run.mjs
```

The workflow writes `out/objects.jsonl`.
