# VizieR Catalog Field Query

[Open this example in StackBlitz](https://stackblitz.com/github/starfetch-js/examples?file=06-vizier-catalog-field-query%2FREADME.md)

Query a small sky field from the VizieR copy of the Gaia DR3 source catalog.

Service: VizieR TAP preset, `vizier`.
Table: `"I/355/gaiadr3"`.

Run:

```sh
node 06-vizier-catalog-field-query/run.mjs
```

The workflow writes `out/vizier-field.jsonl`.
