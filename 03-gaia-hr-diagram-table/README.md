# Gaia HR Diagram Starter Table

[Open this example in StackBlitz](https://stackblitz.com/github/starfetch-js/examples?file=03-gaia-hr-diagram-table%2FREADME.md)

Fetch columns useful for building an HR diagram in a downstream tool such as
Python, TOPCAT, or a notebook. Starfetch only retrieves the table.

Service: ESA Gaia Archive TAP preset, `gaia`.
Table: `gaiadr3.gaia_source`.

Run:

```sh
node 03-gaia-hr-diagram-table/run.mjs
```

The workflow writes `out/hr-diagram-table.csv`.
